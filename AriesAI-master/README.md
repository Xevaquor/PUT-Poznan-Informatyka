Projekt jest próba przeniesienia projektu do srodowiska IntelliJ. Wydaje sie ze dziala. Wlasny gracz jest w klasie QPlayer. Jary z graczami buduja sie do katalogu jars.
Projekt zawiera dwa targety – jeden do odpalania konsolowego runnera (CLI). W jego opcjach musza byc podane nazwy dwóch graczy którzy maja sie zmierzyc ze soba (i musza byc zbudowani do jars).
Drugi target to GUI.
Zmiany:
Wypieprzylem wszystko co zwiazane z powinowactwem watków w projekcie CLI bo nie dzialalo i nie mam zamiaru sie nad tym zanadto pochylac.
